# Return to Olympus

Turn-Based Roguelike RPG based on Greek Mythology

## Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/codetrifecta/return-to-olympus.git
   cd return-to-olympus
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the server:
   ```bash
   npm run dev
   ```
